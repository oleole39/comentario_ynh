Comentario consists of two parts: a comment engine embeded to your web pages and its Administration UI.

It comes with the following features:
- High-level statistics without tracking (but there is an option to enable visitors's IP addresses' collection)
- Multilingual
- Role-based access
- Multiple login options
- Hierarchical comments
- Markdown formatting
- Thread locking
- Sticky comments
- Comment editing and deletion
- Comment voting
- Live comment updates (without page reload)
- Email notifications and RSS feeds
- Multiple domains in one UI
- Flexible moderation rules
- Extensions (antispam in particular)
- Data import/export (from Disqus, Commento, Wordpress / to JSON)

More on https://docs.comentario.app/en/about/features/
